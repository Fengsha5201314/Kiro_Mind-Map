# AI思维导图生成器 - 故障排除指南

## 🚨 "生成思维导图失败" 错误解决方案

### 快速诊断步骤

#### 第一步：使用内置诊断工具
1. 打开应用 → 点击右上角"设置"按钮
2. 在"API密钥配置"部分找到"测试"按钮
3. 点击"测试"按钮，等待诊断完成
4. 查看诊断结果：
   - ✅ 网络连接：正常 - 网络没问题
   - ❌ 网络连接：失败 - 网络连接问题
   - ✅ API密钥：有效 - 密钥正常
   - ❌ API密钥：无效 - 密钥问题

#### 第二步：根据诊断结果采取行动

**如果网络连接失败：**
1. 检查网络连接是否正常
2. 尝试访问 https://google.com 确认网络
3. 关闭VPN或代理软件
4. 检查防火墙设置
5. 尝试使用手机热点

**如果API密钥无效：**
1. 检查API密钥是否正确复制
2. 访问 https://makersuite.google.com/app/apikey
3. 确认密钥状态（是否被禁用或删除）
4. 检查API配额是否已用完
5. 如有必要，创建新的API密钥

### 常见错误代码及解决方案

#### 错误：INVALID_API_KEY
**原因：** API密钥无效或格式错误
**解决方案：**
1. 重新复制API密钥，确保没有多余空格
2. 确认密钥来源正确
3. 检查密钥是否被意外修改

#### 错误：QUOTA_EXCEEDED
**原因：** API调用配额已用完
**解决方案：**
1. 等待配额重置（通常每天重置）
2. 升级Google AI Studio计划
3. 创建新的Google账户和API密钥

#### 错误：NETWORK_ERROR
**原因：** 网络连接问题
**解决方案：**
1. 检查网络连接
2. 尝试刷新页面
3. 关闭其他占用网络的应用
4. 联系网络管理员

#### 错误：TIMEOUT_ERROR
**原因：** 请求超时
**解决方案：**
1. 稍等片刻后重试
2. 简化输入内容
3. 检查网络速度

### 高级故障排除

#### 清除浏览器数据
如果问题持续存在，尝试清除浏览器数据：
1. 按 F12 打开开发者工具
2. 右键点击刷新按钮 → 选择"清空缓存并硬性重新加载"
3. 或者在浏览器设置中清除站点数据

#### 检查浏览器控制台
1. 按 F12 打开开发者工具
2. 切换到"Console"标签
3. 查看是否有红色错误信息
4. 将错误信息截图或复制，便于进一步诊断

#### 尝试不同浏览器
如果问题仍然存在，尝试：
1. 使用Chrome浏览器（推荐）
2. 使用Firefox浏览器
3. 使用Edge浏览器

### 预防措施

#### 定期检查API密钥
1. 每月检查一次API密钥状态
2. 监控API使用配额
3. 备份重要的思维导图数据

#### 保持应用更新
1. 定期刷新页面获取最新版本
2. 清除浏览器缓存
3. 关注功能更新

### 联系支持

如果以上方法都无法解决问题，请提供以下信息：
1. 诊断测试的完整结果
2. 浏览器控制台的错误信息
3. 使用的浏览器类型和版本
4. 网络环境描述
5. 具体的操作步骤

### 应急方案

如果急需使用思维导图功能：
1. 尝试使用不同的网络环境
2. 使用手机热点
3. 尝试在不同时间段使用（避开高峰期）
4. 简化输入内容，分批生成

---

**记住：** 大多数问题都是API密钥或网络连接引起的，使用内置的诊断工具可以快速定位问题！
