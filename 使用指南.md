# AI思维导图生成器 - 使用指南

## 🚀 快速开始

### 1. 访问应用
- **开发服务器地址**: http://localhost:5173/
- **状态**: ✅ 正在运行
- **测试状态**: ✅ 141/141 测试通过
- **构建状态**: ✅ 生产构建成功

### 2. 首次使用步骤

#### 步骤1: 配置API密钥
1. 打开应用后，点击右上角的"设置"按钮
2. 在"API密钥配置"部分输入您的Gemini API密钥
3. 点击"保存"按钮

**获取Gemini API密钥**:
- 访问: https://makersuite.google.com/app/apikey
- 登录Google账户
- 创建新的API密钥
- 复制密钥到应用中

#### 步骤2: 创建思维导图
有三种方式创建思维导图：

**方式1: 文件上传**
1. 点击"上传文件"区域
2. 选择支持的文件格式：
   - PDF文件 (.pdf)
   - Word文档 (.docx)
   - Markdown文件 (.md)
   - 纯文本文件 (.txt)
3. 等待AI分析并生成思维导图

**方式2: 文本输入**
1. 点击"文本输入"标签
2. 在文本框中输入您的内容
3. 点击"生成思维导图"按钮

**方式3: 一键生成**
1. 点击"新建思维导图"按钮
2. 输入主题关键词
3. 选择生成方式
4. 点击"开始生成"

### 3. 编辑思维导图

#### 基本操作
- **拖拽**: 移动节点位置
- **缩放**: 使用鼠标滚轮或缩放按钮
- **平移**: 拖拽空白区域移动视图
- **适应视图**: 点击"适应视图"按钮

#### 节点操作
- **编辑节点**: 双击节点文本
- **添加子节点**: 右键点击节点 → "添加子节点"
- **删除节点**: 右键点击节点 → "删除节点"
- **折叠/展开**: 点击节点上的折叠按钮

### 4. 导出功能

点击"导出"按钮，选择导出格式：
- **PNG图片**: 高质量图片格式，适合分享
- **JSON数据**: 保留完整结构，可重新导入
- **Markdown**: 文本格式，便于编辑

### 5. 历史记录

- **自动保存**: 每次生成的思维导图都会自动保存
- **查看历史**: 点击左侧边栏查看历史记录
- **加载历史**: 点击历史记录项目加载之前的思维导图
- **删除记录**: 点击历史记录的删除按钮

## 🔧 故障排除

### 常见问题

**1. 主题生成失败 - "生成思维导图失败"**

这是最常见的问题，通常由以下原因引起：

*步骤1: 使用内置诊断工具*
1. 打开设置面板
2. 在"API密钥配置"部分点击"测试"按钮
3. 查看诊断结果和建议

*步骤2: 检查API密钥*
- 确保API密钥正确复制，没有多余空格
- 访问 https://makersuite.google.com/app/apikey 确认密钥有效
- 检查API配额是否已用完

*步骤3: 检查网络连接*
- 确保可以正常访问互联网
- 检查防火墙或代理设置
- 尝试关闭VPN后重试

**2. API密钥无效**
- 检查密钥是否正确复制
- 确认API密钥有效且未过期
- 使用"测试"按钮验证密钥状态

**3. 文件上传失败**
- 确认文件格式支持
- 检查文件大小（建议<10MB）
- 尝试重新上传

**4. 网络连接问题**
- 检查网络连接是否正常
- 确认可以访问Google服务
- 尝试使用不同的网络环境

**5. 性能问题**
- 大型思维导图会自动启用虚拟化
- 尝试关闭其他浏览器标签页
- 清除浏览器缓存
